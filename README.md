# Mini_Project

Graph-based financial fraud detection project built in a Jupyter notebook using NetworkX, PyTorch, and GraphSAGE.

The notebook now includes:
- exploratory graphs for class balance, transaction amounts, daily activity, device concentration, and sender outliers
- graph construction and fraud-cluster visualization
- node-level feature engineering with additional feature-detail plots
- model training, ROC and precision-recall diagnostics, and threshold analysis
- a simple risk-monitoring dashboard for the highest-risk accounts

Main file: `fraud_detection_graph_ml.ipynb`
